#Regular Expressions
* "^" = begin
* "$" = end
* [^ ] = not
* [0-9] = matches any number
* [0-9] [0-9] [0-9] = matches any group of 3 digits
* {x} = repetition
** [1-9][0-9]{3} ==> 1000-9999
* "?" = matches 0 or 1 of previous
* "*" = 0 or more
* "+" = 1 or more
* "\d" = any numeric digit
* "\w" = any word; sometimes means whitespace
* [ \t] = whitespace
